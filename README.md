# 📖 Overview

BiharBusiness is an AI-powered hyperlocal business discovery platform built specifically for Bihar.

The Android application enables users to discover businesses, connect directly through WhatsApp, search by city/category, upgrade listings, and manage their business profile.

The app is powered by a Laravel REST API and built using React + Capacitor for native Android deployment.

# 🚀 Setup

```bash
git clone https://github.com/nirajtech15/biharbusiness-app.git

cd biharbusiness-app

npm install

cp .env.example .env

npm run dev
```

---

# 📱 Android Build

```bash
npm run build

npx cap sync android

npx cap open android
```
