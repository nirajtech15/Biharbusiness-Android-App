## Setup

```bash
git clone https://github.com/nirajtech15/biharbusiness-app.git
cd biharbusiness-app
npm install
cp .env.example .env
# .env mein Firebase + API credentials daalo
npm run dev
```

## Android Build

```bash
npm run build
npx cap sync android
npx cap open android
# Android Studio mein Run karo
```
